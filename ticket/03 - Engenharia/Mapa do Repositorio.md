# Mapa do Repositorio

## Pastas principais

- `src/`: app frontend
- `src/components/`: componentes de UI
- `src/pages/`: paginas e entrypoints
- `src/data/`: dados locais e seeds
- `src/server/`: fronteira local usada pelo app
- `server/`: backend real
- `server/http/`: servidor HTTP/BFF
- `server/domain/`: dominios de negocio
- `server/db/`: migracoes e base de dados
- `docs/`: comparacoes, capturas e referencias de produto
- `public/`: assets publicos, mapas e seatmaps
- `scripts/`: utilitarios, geradores e smoke tests

## Documentos chave

- `README.md`
- `roadmap.md`
- `checklist-roadmap.md`
